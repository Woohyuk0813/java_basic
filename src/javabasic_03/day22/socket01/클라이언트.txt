클라이언트가 서버에서 연결요청을 하려면 Socket 객체를 생성하고 생성자 매개값에 서버 IP,Port(필수 값)가 필요하다

Socket socket = new Socket("IP",50001);

클라이언트가 연결요청(Connet()) 을 하고 서버가 연결 수락 (accept()) 했다면
양쪽 socket객체로부터 각각 입력스트림과 출력스트림을 얻을 수 있다.

상대방에게 데이터를 보낼 떄에는 보낼 데이터를 bytep[]배열로 생성하고
매개값으로 해서 OutputStream의 write()메소드를 호출해서 UTF-8 으로 인코딩한 바이트를 배열로 전송한다.

==쓰기 로직==
//클라이언트 -> 서버
String data = "보낼 데이터";
byte[] byte = data.getBytes("UTF-8");
OutputStream os = socket.getOutputStream()
os.write(bytes);
os.flush;

//서버에서 -> 클라이언트
String data = "보낼 데이터";
DataOutputStream dos = new DataOutputStream(socket.getOutputSteam());
os.writeUTF(data);
os.flush();

==읽기 로직 (데이터를 받을 때)==
받은 데이터를 저장할 byte[] 배열 하나 생성하고
이걸 매개값으로 해서 InputStream의 read() 메소드를 호출해서 읽어야지
read() 메소드는 읽은데이터를 byte[] 배열에 저장하고 읽은 바이트 수를 리턴한다.
조건) UTF-8 인코딩

byte[] byte = new byte[1024];

InputStream is = socket.getInputSteam();
int number = is.read(byte);
String data = new String(byte,0,number,"UTF-8");


//단, DataOutputStream 으로 문자열을 보낼때만 DataInputStream으로 읽을 수 있다.
DataInputStream dis = new DataInputStream(socket.getInputStream);
String data = dis.readUTF();
